# AI Art Detector

Forensic tool for spotting AI-generated illustrations and anime art. Runs entirely in your browser - no uploads.

## Methods

- Noise Analysis
- High Pass Filter
- ELA
- Color Banding
- Channel Correlation
- FFT Spectrum
- Brush Patterns
- Impossible Gradients
- Style Coherence

## Usage

Drop an image, hit analyze, check results.

Everything stays on your computer.
